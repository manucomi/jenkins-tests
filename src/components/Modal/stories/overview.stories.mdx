# Modal

The `Modal` component is a React portal that is used to render generic
modal dialogs. The modal also includes a backdrop that can be styled as well
as a focus trap to surround the dialog.

## Use example

```jsx
import Modal from 'mfe-core/ui/Modal';

const [isModalOpen, setIsModalOpen] = useState(false);
return (
    <div>
        {isModalOpen && (
            <Modal
                onClose={handleClose}
                label="Welcome dialog"
                backdropClassName={styles.backdrop}
                modalClassName={styles.modal}
                isFocusTrapped={isModalOpen}
            >
                <button
                    type="button"
                    label="Close this dialog"
                    onClick={handleClose}
                >
                    X
                </button>
                <p>Hello World!</p>
            </Modal>
        )}
    </div>
);
```
