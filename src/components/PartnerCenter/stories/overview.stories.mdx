# PartnerCenter

The `PartnerCenter` component implements the grid of 4 ads.

`Google Publisher Tag (GPT)` is an ad tag library that allows publishers
to define inventory, initiate and bundle ad requests, and render matching demand.
The `PartnerCenter` component requires the script to be loaded in order to function correctly.

## Props

This component provides props for testing purposes only and should not be
taken into account by the consumer.

## Loading GPT script example

Load the GPT library by adding the following to the `<head>` of the HTML document.

```html
<script src="//www.googletagservices.com/tag/js/gpt.js" async="async" />
```

## Usage example

```jsx
// Using default configuration
import PartnerCenter from 'mfe-core/ui/PartnerCenter';

return <PartnerCenter />;
```
