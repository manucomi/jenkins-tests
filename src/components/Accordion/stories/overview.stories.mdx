# Accordion

The `Accordion` component is a wrapper for the `details` and `summary` html elements.

## Props

## Use example

```jsx
import Accordion from 'mfe-core/ui/Accordion';

const title = 'Quick tips for X';

return (
    <Accordion className={styles.accordion} title={title}>
        <ul>
            <li>Do Y</li>
            <li>Do Z</li>
        </ul>
    </Accordion>
);
```
