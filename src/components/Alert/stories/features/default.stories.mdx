# Default Alert

By default the `Alert` is `Polite`.

That is, assistive technologies will notify users of any update, but
do not interrupt the current task.

<Meta
    title="Components/Alert/Features"
    component={Alert}
    argTypes={{
        className: {
            table: { disable: true },
        },
        intrusive: {
            table: { disable: true },
        },
        children: {
            table: { disable: true },
        },
    }}
/>

export function Template(args) {
    const { children, className, active, intrusive } = args;
    return (
        <Alert className={className} active={active} intrusive={intrusive}>
            {children}
        </Alert>
    );
}
