# useConfig Hook

A hook that returns a setter function for each configuration option.
Configuration options are used to tailor mfe-core to an app.

In the table you can see the setters that the hook returns.

<Table
    headings={['Setter function', 'Description']}
    body={[
        [
            '<code>setApiOrigin</code>',
            'Configure the <b>API domain</b> that the services will use for requesting data from the API.',
        ],
    ]}
/>

## Use Example

```js
import useConfig from 'mfe-core/hooks/useConfig';

function App() {
    const { setApiOrigin } = useConfig();
    // set API domain
    setApiOrigin('https://platform.qa.hbr.org');

    return <></>;
}
```
