# commaSeparated

This is a utility function to format an array of strings into a single comma-separated
value.

## Syntax

```js
commaSeparated(arr);
commaSeparated(arr, finalSeparator);
```

## Parameters

<Table
    propsList={[
        {
            name: 'arr',
            type: 'array&#60;string&#62;',
            description: 'The array of strings to format.',
        },
        {
            name: 'finalSeparator',
            type: 'string',
            description:
                'The separator to use before the final item in the list.',
            defaultValue: 'and',
        },
    ]}
/>

## Use example

```js
import commaSeparated from 'mfe-core/utils/commaSeparated';

const list = ['Kathy', 'James', 'Sharon'];

console.log(commaSeparated(list)); // Kathy, James, and Sharon
console.log(commaSeparated(list, 'or')); // Kathy, James, or Sharon

list.pop();
console.log(list); // ['Kathy', 'James']
console.log(commaSeparated(list, 'or')); // Kathy or James
```
