# Tiempos Headline

Tiempos Headline is designed for larger headline sizes, striking a balance between practicality and elegance.

## Import Font

Import the Tiempos Headline font styles into one of your SASS modules.

```css
// _fonts.scss

@import 'mfe-core/typography/tiempos-headline.css';
```

## Use font family

```css
.element {
    font-family: 'Tiempos Headline';
}
```

export function Template(args) {
    const { weights, size, text, fontFamily, italic } = args;
    return (
        <FontHelper
            text={text}
            size={size}
            fontFamily={fontFamily}
            weights={weights}
            italic={italic}
        />
    );
}
