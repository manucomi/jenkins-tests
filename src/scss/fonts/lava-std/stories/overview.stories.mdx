# Lava Std

Lava was designed for magazine use. It can handle large quantities of text
with ease. It's legible and harmonious at small sizes, sophisticated and elegant at large sizes.

## Import Font

Import the Lava Std font styles into one of your SASS modules.

```css
// _fonts.scss

@import 'mfe-core/typography/lava-std.css';
```

## Use font family

```css
.element {
    font-family: 'Lava Std';
}
```

export function Template(args) {
    const { weights, size, text, fontFamily, italic } = args;
    return (
        <FontHelper
            text={text}
            size={size}
            fontFamily={fontFamily}
            weights={weights}
            italic={italic}
        />
    );
}
